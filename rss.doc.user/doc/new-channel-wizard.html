New Channel Wizard

This wizard allows you to create a new channel from an external RSS feed and save it as a file in the workspace. The wizard retrieves the initial channel content from the feed URL. You must select a translator suitable for the particular feed. Optionally, you may specify, in minutes, the interval at which the channel's content will be refreshed from its source URL.

Using Auto-discovery

A variant of this wizard is available, which uses standard auto-discovery methods to detect links to syndicated feeds in a web page. The initial page will prompt you to enter a URL of a web page, in which you wish to auto-detect syndicated feeds. After retrieving the page, a list of feeds published in the web page is displayed. You may then select a feed and proceed to create a channel in your workspace.
