New Channel Wizard

This wizard allows you to create a new channel from an external RSS feed and save it as a file in the workspace. The wizard retrieves the initial channel content from the feed URL. You must select a translator suitable for the particular feed. Optionally, you may specify, in minutes, the interval at which the channel's content will be refreshed from its source URL.
