Eclipse RSS Reader Preferences

The RSS preferences page allows you to specify the default channel update interval and select the default link browser, which are used when creating new channels. In addition, the included link browsers (help and editor) have their own preference pages; the former allows you to select a configured help browser, and the latter an editor to use for browsing channel items.
