Download

Before downloading the Eclipse RSS Reader, please make sure you have the following pre-requisites:

  • Eclipse 2.1 with JRE 1.4
    -OR-
    IBM WebSphere Studio Application Developer, v5.1
  • Web browser with Internet connection

To install the Eclipse RSS Reader into your workbench using Eclipse, point your Update Manager to http://pnehrer.com/rss. Alternatively, you can download the latest feature archive and unzip it into your Eclipse product installation directory (i.e., the one that contains the main eclipse subdirectory).

This software has been developed and tested on Slackware Linux 9.1 w/ GTK, and MS Windows 2000.
