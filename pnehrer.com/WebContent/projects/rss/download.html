Download

There are two ways to install the Eclipse RSS Reader:

  1. Download the appropriate feature archive and unzip it into your Eclipse product installation directory.
  2. Use your Update Manager to install the appropriate Eclipse feature.

Two target environments are supported:

Eclipse 2.1

This option is suitable for all applications based on Eclipse 2.1. Download com.pnehrer.rss_1.2.1.zip and unzip it into your Eclipse product installation directory (i.e., the one that contains the eclipse subdirectory). Restart your workbench. Alternatively, use your Update Manager to download and install the Eclipse RSS Reader feature.

IBM WebSphere Studio Application Developer v5.1 for MS Windows

This option is suitable specifically for the Windows version of WSAD. Download com.pnehrer.rss.wsad_1.2.1.zip and unzip it into your WSAD installation directory. Restart your workbench. Alternatively, use your Update Manager to download and install the Eclipse RSS Reader for WSAD feature. Note that as of this release, it is no longer necessary to install the Eclipse RSS Reader feature before installing the WSAD-specific extensions.

This software has been developed and tested on Slackware Linux 9.1 with GTK, as well as MS Windows 2000. Note that this version has NOT been developed for, or tested with, Eclipse 3.0.
