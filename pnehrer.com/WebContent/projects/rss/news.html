2003-12-29: Eclipse RSS Reader v1.0.0 released
Initial release of the Eclipse RSS Reader has been published. The features include an Eclipse-based uniform channel model, periodic updates from external source, property and preference pages, New Channel wizard as well as a Channel Navigator and Detail views. In this release the reader supports RSS versions 0.90, 0.91, 0.92, 1.0, and 2.0.
