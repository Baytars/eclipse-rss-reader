Castor Plug-in for Eclipse

The goal of this project is to provide support for the Castor framework and tools within the Eclipse platform. Some of the project's short-term objectives include:

  1. Exposing Castor Source Generator as an incremental project builder.
  2. Providing a visual editor for Castor's XML Binding files, which are used during generation of XML marshalling code.
  3. Providing a visual editor for Castor's XML Mapping files, which are used during marshalling and demarshalling of Java objects to and from XML.

Please visit the project's News page to learn about the latest developments.
