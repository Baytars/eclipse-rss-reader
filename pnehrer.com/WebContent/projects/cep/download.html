Download

com.pnehrer.castor_1.2.1.zip -- plug-in installation archive. To install/upgrade:

  1. Uninstall previous versions of the plugin (i.e., remove the com.pnehrer.castor_* directories in your Eclipse plugins directory.)
  2. Extract the ZIP file into your Eclipse's plugin directory.
  3. Restart Eclipse.
This release has been developed and tested on Linux 2.4.21 with Sun's JDK 1.4.2 and Eclipse 2.1.1. It has been verified to be compatible with WSAD 5.0.1 (and thus IBM's JDK 1.3) running on Windows 2000.
