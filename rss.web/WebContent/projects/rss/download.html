Download

Before downloading the Eclipse RSS Reader, please make sure you have the following pre-requisites:

  • Eclipse 2.1 with JRE 1.4
    -OR-
    IBM WebSphere Studio Application Developer, v5.1
  • Web browser with Internet connection

You can use the Eclipse Update Manager to install the Eclipse RSS Reader into your Workbench. Alternatively, you can download the latest feature archive and unzip it into your Eclipse product installation directory (i.e., the one that contains the main eclipse subdirectory). If you're using WSAD in MS-Windows you may also download an extension that will allow you to view the links in a web browser editor instead of an external browser.

This software has been developed and tested on Slackware Linux 9.1 w/ GTK, and MS Windows 2000.
