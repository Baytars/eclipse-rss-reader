Update Site

This page will allow you to install the Eclipse RSS Reader into your workbench using the Update Manager. However, you must do the following to make it work:

  1. Make sure the Web-triggered updates feature is enabled in your workbench Preferences. To enable this feature, open the Preferences by choosing to Window -> Preferences, then select Install/Update -> Web-triggered Updates. Check Accept Internet connections for Web-triggered updates.
  2. View this page from within the Update Manager. To do that, start the Update Manager by choosing Help -> Software Updates -> Update Manager. Create a new web bookmark and point it to http://morphine.sourceforge.net/updates. In the preview window, click the Site home page link.

To get started, please choose a SourceForge mirror to download from:
