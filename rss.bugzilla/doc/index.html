Bugzilla RSS Plugins

This plugin provides simple translators for Bugzilla query result and bug detail pages. This is useful if you want to keep track of any new bugs that may match certain query criteria, as well as any additional comments added to a specific bug report. It works with Bugzilla versions that support XML-based query results and bug details.

Query Results

To set up a channel based on Bugzilla query results, start with the results page's URL, then append "&ctype=rdf" to the end of it. This will cause the results page to be returned in the RDF format, which is recognized by this plugin. If you want to provide a custom title for the channel (the query results don't come with any descriptive titles by default), append "&title=<YOUR+TITLE>&ctype=rdf" to the end of the URL instead. Be careful to properly URL-encode the title parameter.

E.g., https://bugs.eclipse.org/bugs/buglist.cgi?query_format=specific&order=relevance+desc&bug_status=__open__&product=&content=rss&title=All+rss&ctype=rdf

Bug Detail

To set up a channel based on a Bugzilla bug detail page, start with the page's URL, then replace the file part (show_bug.cgi) with xml.cgi. This will cause the URL to return the bug detail data in XML format, which is recognized by this plugin.

E.g., https://bugs.eclipse.org/bugs/xml.cgi?id=83885
